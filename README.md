# Comunicaciones

### Comandos

* Clonar un Repositorio

```console
$ git clone https://github.com/afbaeza/Comunicaciones.git
$ cd Comunicaciones
$ npm install
```

* Subir cambios al repositorio

```console
$ git status
$ git add modifiedFile
$ git status
$ git pull
$ git commit -m "Descripcion del commit"
$ git push -u origin master
```
